# HTML Editor

A lightweight HTML editor built with Monaco Editor, featuring live preview, syntax highlighting, and HTML validation.

## Project Structure

The project consists of three main files:

- `index.html`: The main HTML structure and layout.
- `script.js`: JavaScript logic for editor functionality and live preview.
- `styles.css`: Custom styles for the editor and preview pane.

## Getting Started

1. **Clone the Repository**

   ```bash
   git clone https://github.com/imamulkadir/htmlEditor.git
   cd htmlEditor

   ```

2. **Open** `index.html`
   Open the `index.html` file in your preferred browser to launch the editor.

3 **Features**

- Monaco Editor Integration: Provides a rich code editing experience.
- Live Preview: See your HTML changes in real-time.
- HTML Validation: Highlights errors in your HTML code.
- Click-to-Editor Sync: Click on elements in the preview to jump to the corresponding code in the editor.

4. **Notes**

- Ensure you have an active internet connection, as Monaco Editor and other dependencies are loaded via CDN.
- The editor supports basic HTML5 elements and attributes.
